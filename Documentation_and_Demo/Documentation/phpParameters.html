List of PHP Configuration Parameters

ParameterUse
$db
Mandatory

Database Manager that you'll use in your project


Default Value: none
Format: "mysql"|"sqlserver"|"postgres"|"sqlite"
Examples: 
$db="mysql"; | $db="sqlite";
$host
Mandatory (except for sqlite)

Host where is placed the database


Default value: none
Format: "host"
Examples: 
$host="localhost";
$port
Mandatory (only for postgresql)

For PostgreSQL you must specify the port that is using. Usually it use 5432 as default


Default value: none
Format: "#Port"
Examples: 
$port="5432";
$dbname
Mandatory

Name of the database that you will use. For sqlite this parameter contains the relative path (from the "easyTables/php/" directory) of the database file.


Default Value: none
Format: "databaseName" (for sqlite "relatiPath/databaseFileName")
Examples: 
$dbname="dataBaseName"; | $dbname="../../databaseFileName";
$table
Mandatory (except if you use instead $query and won't use the insert, update or delete options)

Name of the table source of information. You can use the parameter $query and use a sql query as source of information, but if you will use the option to insert, update and delete information this field will be mandatory.


Default Value: none
Format: "tableName"
Examples: 
$table="books"; | $table="employees"; | $table="emails";
$user
Mandatory (except for sqlite)

Database accredited user name


Default Value: none
Format: "userName"
Examples: 
$user="root"; | $user="administrator";
$pass
Mandatory (except for sqlite)

Database accredited user password


Default Value: none
Format: "password"
Examples: 
$pass="pwAdm46";
$fields
Optional

Array of table fields that will be shown. If you don't use this parameter by default it will show all the fields of the selected table.


Default Value: none
Format: array("field1","field2", ...)
Examples: Working Example
$fields=array("name", "birthday", "email");
$colNames
Optional

Use this parameter if you want to change one or several field names that will be displayed in the table header


Default Value: none
Format: array("field1"=>"newName1", "field2"=>"newName2")
Examples: Working Example 
        $colNames=array(
            "product"=>"Product Name", 
            "p_cost"=>"Cost"
            );
$colSizes
Optional

Use this parameter when you want to set a specific width for one or several columns.


Default Value: none
Format: array("field1"=>"width", "field2"=>"width")
Examples: Working Example 
        $colSizes=array(
            "field1"=>"50px",
            "field2"=>"30%",
            "field3"=>"200em"
            );
$query
Optional

You can use this parameter as source of information instead of $table. This parameters consists of a sql query. If you will use the insert, update and delete options it will be necesary use the parameter $table as well.


Default Value: none
Format: "select sql query"
Examples: Working Example 1 o Working Example 2
$query="select * from books";
$csvChar
Optional

Use this parameter to change the default csv character. This apply for the CSV export option.


Default Value: ","
Format: "character"
Examples: Working Example
$csvChar=";"; | $csvChar=":";
$fileName
Optional

Use this option to change the default name of the generated exporter file.


Default Value: "EasyTables"
Format: "desiredFileName"
Examples: Working Example
$fileName="Books"; | $fileName="Products";
$debugMode
Optional

When you enable this parameter, the sql queries that are being executed will be shown.


Default Value: false
Format: true|false
Examples: Working Example
$debugMode=true;
$friendlyOutput
Optional

Use this parameters to replace a determinated string by an html element in a column, for example a picture.


Default Value: none
Format: array("field"=>array("string"=>"replacement"))
Examples: Working Example 
        $friendlyOutput=array(
            'in_stock'=>array(
                '0'=>'',
                '1'=>''
                )
            );
$colColorsOdd
Optional

Use this parameter to change the default color of the odd rows of a column.


Defautl Value: none
Format: array("field"=>"#hexcolor")
Examples: Working Example 
        $colColorsOdd=array(
            "id"=>"#E38F33",
            "price"=>"#E38F33"
            );
$colColorsEven
Optional

Use this parameter to change the default color of the even rows of one or several columns.


Default Value: none
Format: array("field"=>"#hexcolor")
Examples: Working Example 
        $colColorsEven=array(
            "id"=>"#EDCF76",
            "price"=>"#EDCF76"
            );
$colFontColors
Optional

Use this parameter to change the default font color of one or several columns.


Default Value: none
Format: array("field"=>"#hexcolor")
Examples: 
        $colFontColors=array(
            "id"=>"#BBAAFF",
            "price"=>"#AAFFBB"
            );
$colHeadColors
Optional

Use this parameter if you want to change the default background header color of one or several columns.


Default Value: none
Format: array("field"=>"#hexcolor")
Examples: Working Example 
        $colHeadColors=array(
            "id"=>"#CF0707",
            "price"=>"#CF0707",
            "name"=>"#1527CA"
            );
$colOrderColors
Optional

With this parameter you can change the default background color of the active sort column.


Default Value: none
Format: array("Even"=>"#hexcolor", "Odd"="#hexcolor")
Examples: Working Example 
        $colOrderColors=array(
            "Even"=>"#7CE1F3",
            "Odd"=>"#3ECAF9"
            );
$colOrderFontColors
Optional

With this parameter you can change the default font color of the active sort column.


Default Value: none
Format: array("Even"=>"#hexcolor", "Odd"="#hexcolor")
Examples: Working Example 
        $colOrderFontColors=array(
            "Even"=>"#000",	
            "Odd"=>"#000"
            );
$colHeadFontColors
Optional

Use this parameter to change the default header font color of one or several columns.


Default Value: none
Format: array("field"=>"#hexcolor")
Examples: 
        $colHeadFontColors=array(
            "id"=>"#000000",
            "price"=>"#FFFF00"
            );
$colAlign
Optional

Change the default aligment of one or several columns.


Default Value: none
Format: array("field"=>"aligment")
Examples: Working Example 
        $colAlign=array(
            "id"=>"right",
            "price"=>"center"
            );
